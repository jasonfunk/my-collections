# Project Overview

## Description

The project itself is both a learning project and a full development project. The result of the project will be to create a personal website and mobile application to track several of my collections (original Star Wars action figures, G1 Transformers, original He-Man action figures, etc.). All of this will be done using Claude Code.

## Goals

I have several goals for this project:

1. Learn Claude Code from the ground up with a practical application in multiple layers of the tech stack
	1. Claude code project structure
	2. Common plugins and agents
	3. Update CLAUDE.md and other context files for maximum efficiency and usefuless
2. Learn OAuth2
	1. Have OAuth2 be the auth method for all interfaces and APIs
	2. Set up OAuth2 from scratch
3. Learn React
	1. UI will use React
4. Improve my Typescript and Javascript (ES16)
	1. I'm an old school Javascript developer (pre-ES5)
	2. The developers I support use Typescript at work
	3. I want the APIs and underlying code to use TypeScript
4. Learn mobile application development
	1. Android first
	2. IOS second
5. Full SDLC control
	1. Jira project tracking
	2. Confluence documentation
	3. CI/CD pipeline
6. Have a useable deployed solution at the end
